# California Driver License / ID — Expert Forensics Briefing

## Near-certain fraud indicators

Any single failure here effectively confirms forgery: **DL number not matching exactly 1 uppercase letter + 7 digits** (e.g. "A1234567"); barcode header not beginning **"@636014"** (California's AAMVA IIN); **QR code** on back instead of **PDF417**; any front-vs-barcode field mismatch; **"DRIVER'S LICENSE"** with an apostrophe (genuine label is **"DRIVER LICENSE"**, no apostrophe); an under-21 holder on a horizontal card (must be vertical/portrait).

## Specific forger mistakes

- Eye color as full words ("Brown") instead of AAMVA 3-letter codes: **BRN, BLU, GRN, HZL, GRY, BLK**
- Wrong color for type text: **DL type must be BLUE; ID type must be GREEN** (2018+)
- Dates in any format other than **MM/DD/YYYY**; ISS ≥ EXP; EXP not ~5 years after ISS
- Names not in **UPPERCASE**, or last name not on line 1, first/middle on line 2
- Post-Jan 2018 card missing BOTH the **gold-bear/white-star REAL ID icon** AND **"FEDERAL LIMITS APPLY"** — exactly one must appear
- Sex field anything other than **M, F, or X**

## Cross-field barcode traps

Forgers routinely alter the front but neglect the PDF417. Verify these AAMVA fields against the front: **DAQ** (license number), **DCS/DAC/DAD** (last/first/middle name), **DBB** (DOB), **DBA** (expiration), **DBD** (issue), **DAU** (height), **DAY** (eye color), **DBC** (sex: **1=M, 2=F, 9=X**), **DCF** (Document Discriminator — must also match printed DD on back), and **DAJ** (state, must equal **"CA"**). A single mismatch is definitive tampering.

## Generation identification

- **Gen 4 (Oct 2025+)**: redwood/poppies/Pacific-coastline background, **NO magnetic strip**, **TWO barcodes on back** (second carries digital signature)
- **Gen 3 (2018–Sep 2025)**: Gold Rush miner + poppies background, **magnetic strip present**
- Gen 4 with a magnetic strip, or Gen 3 missing one, is a generation-mismatch red flag. REAL ID bear+star applies to both generations.

## Visual security features (trickiest checks)

The **DOB is laser-engraved in 3D tactile numerals across the primary photo** — flat ink-printed DOB is a strong forgery tell. **Laser-perforated bear outline** is visible only when the card is backlit. Under **UV light**, expect a **full-color ghost image of the cardholder** plus hidden patterns; absence indicates counterfeit. Holographic laminate must show **CA-themed imagery (bear, state seal)** that shifts when tilted — generic rainbow holograms indicate overlay forgery. **Microprinting** around the photo ("CA DMV" / "VALID ID") should remain crisp under magnification; blurry or dotted microprint is a print-shop counterfeit.
